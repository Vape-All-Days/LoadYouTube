<h2 align="left">Installation</h2>

###

<h4 align="left">Setup</h4>

###

```
git clone https://github.com/iamironman0/youtube_downloader.git
```
```
cd youtube_downloader
```
```
python setup.py
```
and Finally run it
```
python main.py
```
